## DSA Repository

This repository contains implementations of various Data Structures and Algorithms (DSA) in Python.

### Structure

- `arrays/`: Array-based problems and implementations.
- `linked_list/`: Singly and Doubly Linked List implementations and problems.
- `stack_queue/`: Stack and Queue implementations and problems.
- `tree/`: Tree data structures (Binary Trees, BSTs) and problems.
- `graph/`: Graph algorithms and problems.
- `dynamic_programming/`: Dynamic Programming solutions.
- `greedy/`: Greedy algorithm solutions.
- `sorting/`: Sorting algorithms.
- `searching/`: Searching algorithms.
- `backtracking/`: Backtracking solutions.
- `heap/`: Heap data structures and problems.
- `trie/`: Trie data structure and problems.
- `matrix/`: Matrix-based problems.
- `string/`: String manipulation problems.
- `math/`: Mathematical algorithms.
- `bit_manipulation/`: Bit manipulation techniques.
- `recursion/`: Recursive solutions.
- `design/`: Design pattern problems.
- `interview_questions/`: Common interview questions.
- `dsa_sheet/`: Solutions to DSA sheets (e.g., LeetCode, GeeksforGeeks).
- `utils/`: Utility functions and helpers.

### Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd dsa_dev
   ```

2. Explore the directories to find the data structure or algorithm you want to study.

3. Each file contains implementations and examples of how to use them.
